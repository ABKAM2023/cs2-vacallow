## RU
**VacAllow** - плагин пропускает игроков с VAC-блокировкой, даже когда VAC включён.

**Требования:**
- [Utils](https://github.com/Pisex/cs2-menus/releases)

## EN
**VacAllow** - the plugin allows players with VAC bans to join even when VAC is enabled.

**Requirements:**
- [Utils](https://github.com/Pisex/cs2-menus/releases)
